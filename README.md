# 全台A1、A2車禍資料處理
### 使用資料
* [內政部警政署歷史交通事故資料](https://www.npa.gov.tw/NPAGip/wSite/lp?ctNode=12854&CtUnit=2633&BaseDSD=7&mp=1 "link")  
  > * A1類交通事故係指造成人員當場或24小時內死亡之案件  
  > * A2類交通事故則是指造成人員受傷或24小時後死亡之案件
